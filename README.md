Credit Calculator (Кредитный калькулятор)

Проект для расчёта кредитных платежей с поддержкой двух типов графиков.

Возможности:
- **Аннуитетные платежи** (одинаковые каждый месяц)
- **Дифференцированные платежи** (уменьшаются к концу срока)
- **Первоначальный взнос** (учитывается при расчёте)
- **Валидация ввода** (защита от некорректных данных)
- **Расчёт переплаты** (показывает общую стоимость кредита)



Пример использования
```java
AnnuityCalculator calculator = new AnnuityCalculator();
List<Payment> schedule = calculator.calculate(100000, 10, 2, 20000);
